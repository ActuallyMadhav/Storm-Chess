# Storm-Chess
Creating my own chess engine - Storm. A project to dive into the world of chess programming

## Methodology
### UI
will use opengl as the main graphics api. displaying the board and the user interface will be done via opengl as well

### Backend
going to implement 2 search algorithms: 
i) MiniMax with Alpha-Beta pruning
ii) Monte-Carlo Search Tree

## Goal
my goal is to create a chess engine that beat me and hopefully tons of actually strong players. hopefully with an elo of approximately 2000
